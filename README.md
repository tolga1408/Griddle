# Griddle

A single-file browser word game built with pure HTML, CSS, and vanilla JavaScript.

## What is included

- `index.html` contains the full game, styling, and JavaScript.
- No backend is required.
- No npm or build step is required.
- The game can run locally or on GitHub Pages.

## Features

- 5 by 5 waffle-style word grid
- Swap letters by clicking two tiles
- 15 swap limit
- Daily date-seeded puzzle
- Color feedback for each tile
- Mobile responsive layout
- Dark mode toggle
- First-visit help modal
- Local win streak using localStorage
- Emoji-grid share button

## Run locally

1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Play the game.

## Deploy on GitHub Pages

1. Open the repository on GitHub.
2. Go to Settings.
3. Click Pages in the sidebar.
4. Set Source to Deploy from a branch.
5. Select branch `main` and folder `/root`.
6. Click Save.
7. After GitHub finishes publishing, open:

```text
https://tolga1408.github.io/Griddle/
```

## Updating

Edit `index.html`, commit the change, and GitHub Pages will update after the commit reaches `main`.
